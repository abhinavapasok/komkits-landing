Scene box: composition never changes.

Wiring is ~10 lines: wrapper gets aspect-ratio: 340/300, the scene inside stays a fixed 340×300 with transform-origin: top left, and a ResizeObserver sets --s = wrapperWidth / 340. Meta text below the scene stays fluid on purpose — only the authored composition scales. Trade-off: text inside the scene shrinks below UA font sizes at phone widths, which is fine for decorative scenes (06's stages, 08's diagram) but the two playable cards (07 virtual-cursor, 10 dashboard editor) should floor the scale (e.g. max(--s, 0.8) + allow horizontal pan) or keep fluid layout below a breakpoint so tap targets stay ≥44px.
